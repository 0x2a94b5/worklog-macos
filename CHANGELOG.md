# Changelog

本项目遵循 [Semantic Versioning](https://semver.org/)。

## [1.0.0] - 2026-07-14

### Added

- 原生 SwiftUI 与 AppKit macOS 应用，最低支持 macOS 12
- 月份、分类及未完成/已完成状态筛选
- 主任务与子任务、折叠和同级拖动排序
- 单击选择、双击编辑及键盘导航
- 全局搜索与任务定位
- Markdown 追加导入和标准 `.md` 文件导出
- SQLite 本地存储、版本迁移、事务和滚动备份
- 删除后的会话级撤销恢复
- 月度复盘生成与复制
- 核心回归测试和 GitHub Actions 构建检查
